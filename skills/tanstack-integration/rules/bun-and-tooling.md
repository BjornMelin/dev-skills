# Bun and Tooling

Follow repo package-manager policy while quoting official docs carefully.

## Rules

- In Bun-first repos, use `bun run`, `bunx --bun`, `bun add`, and `bun install` for custom commands.
- Do not rewrite official docs quotes just to use Bun.
- Use `tanstack search-docs ... --json` and `tanstack doc ... --json` when the TanStack CLI is installed.
- Do not recommend the removed TanStack CLI MCP helper command.
